pushContext(message, user, channel, type) → {Promise}

Pushes a new context with a given message, user, channel and type. This discards any other context of the same type for the same user/channel combination.

Parameters:
Name Type Description
message String

Message to be sent to the target user in the target channel.
user User | String

Target user to receive the message. Can either be an User instance or the ID used by the Adapter to identify the target user.
channel Channel | String

Channel to where the message will be sent to. Can either be a Channel instance, or the ID used by the Adapter to identify the target channel.
type Integer

Type of context to be pushed. Types are defined in the Context model and can be Context.NUMBER, Context.BOOLEAN, or Context.REGEX. Documentation about those items and how they behave can be found in the Context documentation.
